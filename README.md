# 📦 Estoque API

API REST para gestão de estoque e pedidos com inteligência artificial integrada.

> 🚧 Projeto em construção

---

## 🚀 Tecnologias

- Java 21
- Spring Boot 4.x
- Spring Security + JWT
- Spring Data JPA + Hibernate Envers
- PostgreSQL
- MapStruct
- Bucket4j
- Spring AI + OpenAI
- Swagger / OpenAPI
- Docker

---

## 📋 Módulos planejados

- ✅ Models e Repositories
- ⏳ Autenticação com JWT + Refresh Token
- ⏳ Gestão de Categorias
- ⏳ Gestão de Fornecedores
- ⏳ Gestão de Produtos
- ⏳ Gestão de Pedidos
- ⏳ Movimentações de Estoque
- ⏳ Auditoria com Envers
- ⏳ Rate Limiting com Bucket4j
- ⏳ Inteligência Artificial com Spring AI
- ⏳ Deploy com Docker + Oracle Cloud
